Ubuntu Single User Mode

有段时间没有发文了，发一炮。
怎么进入单用户模式呢？

  • 1、开机，按Shift进入grub启动菜单。
  • 2、选择Recovery Mode，按e，进入选单编辑。
  • 3、将'ro.*$‘替换为’rw single init=/bin/bash'，按ctrl+x进入启动。
  • 4、启动后，即进入单用户模式。

这个东西，每次用都要搜索，这次保存下来。
